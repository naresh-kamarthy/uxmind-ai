# UXMind — High-Fidelity Visual Artifacts

This document maintains the canonical reference and technical proofs for the generated visual assets powering the UXMind ecosystem. All raw images reside strictly in `src/assets/`.

## The Obsidian Glass Aesthetic
All assets conform to the project's **Minimalist Cyberpunk / Obsidian Glass** aesthetic, characterized by:
- 40px Backdrop Blur
- 1px Hairline Borders (Opaque & Semi-transparent variations)
- Deep, desaturated core color palettes enhanced with neon accents.
